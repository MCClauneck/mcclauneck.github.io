MCSlayerRewards

Change log

This page always mirrors the latest MCSlayerRewards release. Every version also has a permanent link in the version index.

1.0.0 Latest 2026-07-20
Added
  • Per-mob money rewards — a fixed value or a random min-max range in any MCEconomy currency.
  • Custom Base64 item drops with per-drop chances, and an option to cancel vanilla drops.
  • Holographic +amount currency feedback at the kill location.
  • In-game drop editor via /slayerrewards edit <mob> (alias /slayer).
  • Five storage backends for kill/reward logs — SQLite, MySQL, PostgreSQL, MongoDB, and the central REST server.
  • A universal engine jar that runs on SpigotMC, PaperMC, and FoliaMC.
Changed
  • Rebuilt from an MCEconomy extension into a standalone multi-module plugin (MCSlayerRewards), with the namespace io.github.mcclauneck.slayerrewards.
Notes
  • Depends on MCEconomy and MCIdentity; both are listed in the depend block of plugin.yml.
