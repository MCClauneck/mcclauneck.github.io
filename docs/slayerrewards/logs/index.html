MCSlayerRewards

Change log

This page always mirrors the latest MCSlayerRewards release. Every version also has a permanent link in the version index.

1.1.0 Latest 2026-07-21
Changed
  • Kill rewards are now paid to the killer's active MCIdentity identity, so money is kept per character.
  • Reward currencies are now the currencies configured in MCEconomy, matched by their system code. The mob-drop editor cycles through whatever currencies the server has instead of a fixed list.
  • Updated the plugin to the current MCEconomy provider API.
Added
  • Payouts now honor MCEconomy's transaction veto and announce a wallet update, so other plugins can block or react to rewards.
Notes
  • Kill and reward logs remain keyed by the player, so per-player statistics are unchanged.
  • Requires MCEconomy and MCIdentity; both are listed in the depend block of plugin.yml.
