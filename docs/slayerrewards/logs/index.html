Change log

This page always mirrors the latest MCSlayerRewards release. Every version also has a permanent link in the version index.

1.1.1 Latest 2026-07-21
Changed
  • Fixed the plugin so it builds correctly against the MCEconomy host API: its Maven group is now distinct from the host's, so the MCEconomy artifacts resolve instead of being shadowed by the plugin's own modules.
Notes
  • No gameplay or configuration changes from 1.1.0 — the reward payouts, identity-scoped money, and MCEconomy currencies are unchanged.
  • Requires MCEconomy and MCIdentity; both are listed in the depend block of plugin.yml.
