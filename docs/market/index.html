MCMarket

A standalone, GUI-based shop for the MCEconomy ecosystem. Create unlimited markets, edit them with a drag-and-drop in-game editor, and trade items for any MCEconomy currency.

io.github.mcclauneck.market SemVer 1.0.0 depends: MCIdentity, MCEconomy
Change log Repository ↗

Highlights

🛒
Infinite markets. Create unlimited categories — Ores, Food, Blocks — each a plain YAML file.
🖱️
In-game editor. Drag and drop items to set buy/sell prices; no config editing required.
⚡
Asynchronous. Transactions run off the main thread, and payouts flow through the MCEconomy provider.
🧬
Base64 storage. Every item detail — enchantments, NBT, durability — is preserved.
🗄️
Five storage backends. SQLite (default), MySQL, PostgreSQL, MongoDB, or the central REST server.

Module architecture

A multi-module Gradle build modelled on MCEconomy. The universal engine jar shades every module and the thin platform entry points into one plugin.

ModuleJava packageResponsibility
apiio.github.mcclauneck.market.apiPure-Java contracts: the data-store interface and the item transport record.
common…market.commonThe SQLite/MySQL/PostgreSQL/MongoDB backends and the central REST client.
bukkit…market.bukkitShared Bukkit core: provider, drag-and-drop editor, the /market command, and the inventory listener.
spigotmc / papermc / foliamc…market.<platform>Thin per-platform entry points.
engine…market.engineThe universal jar for SpigotMC, PaperMC, and FoliaMC.

Command

/market <name> [page]
Open a market's shop GUI.
/market create <name>
Create a new market and open its editor (permission market.admin).
/market edit <name> [page]
Open the drag-and-drop editor for a market (permission market.admin).
🔗

MCMarket runs alongside MCEconomy and MCIdentity. It declares both in the depend block of its plugin.yml, so they load first, and buys and sells settle through the MCEconomy economy provider.
