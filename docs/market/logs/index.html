Change log

This page always mirrors the latest MCMarket release. Every version also has a permanent link in the version index.

1.1.0 Latest 2026-07-21
Changed
  • Buying and selling now settle against your active MCIdentity identity, so money is kept per character — equip an identity before trading.
  • Item currencies are now the currencies configured in MCEconomy, matched by their system code. The market editor cycles through whatever currencies the server has instead of a fixed list.
  • Updated the plugin to the current MCEconomy provider API.
Added
  • Purchases and sales now honor MCEconomy's transaction veto and announce a wallet update, so other plugins can block or react to market trades.
  • A sale returns your item automatically when the payout cannot be completed.
Notes
  • Requires MCEconomy and MCIdentity; both are listed in the depend block of plugin.yml.
