Version 1.0.0

A permanent link to the MCMarket 1.0.0 release notes.

1.0.0 Latest 2026-07-20
Added
  • GUI shop with unlimited markets, each defined by a plain YAML file.
  • Drag-and-drop in-game editor via /market create and /market edit.
  • Base64 item storage that preserves enchantments, NBT, and durability.
  • Five storage backends — SQLite, MySQL, PostgreSQL, MongoDB, and the central REST server.
  • A universal engine jar that runs on SpigotMC, PaperMC, and FoliaMC.
Changed
  • Rebuilt from an MCEconomy extension into a standalone multi-module plugin (MCMarket), with the namespace io.github.mcclauneck.market.
Notes
  • Depends on MCEconomy and MCIdentity; both are listed in the depend block of plugin.yml.
