MCClauneck ecosystem

A per-identity economy for Minecraft servers

MCEconomy is a multi-currency, multi-platform economy plugin built on MCIdentity, where every identity is its own account. MCMarket and MCSlayerRewards extend it with a GUI shop and mob-slaying rewards.

How it fits together

One identity, one account

💡

MCEconomy binds balances to the active MCIdentity identity, not the player. MCMarket and MCSlayerRewards are standalone plugins that run alongside MCEconomy and MCIdentity and pay out through the same economy provider, so every currency stays consistent across the server.
