Version 1.0.0

A permanent link to the MCEconomy 1.0.0 release notes.

1.0.0 Latest 2026-07-20
Added
  • Multi-currency economy with per-currency transferability (for example coin, gold, and account-locked point).
  • Per-identity accounts: balances are keyed to the active MCIdentity identity.
  • /balance GUI and /pay <player> <currency> <amount> transfers.
  • Five storage backends — SQLite, MySQL, PostgreSQL, MongoDB, and the central REST server.
  • A universal engine jar that runs on SpigotMC, PaperMC, and FoliaMC, detected at enable time.
Changed
  • Standardized the Java namespace to io.github.mcclauneck.mceconomy; published Maven coordinates remain io.github.mcclauneck:<module>.
Notes
  • Requires MCIdentity on the server; declare it in depend for correct load order.
